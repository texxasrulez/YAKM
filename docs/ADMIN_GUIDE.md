# Admin Guide

- **Login**: `/kontact/admin/login.php`
- **Dashboard**: quick links.
- **Messages**: view/search; delete individual rows; CSV export.
- **Settings**: site info, theme, SMTP, recipients JSON, auto‑reply toggle/template, min submit seconds.
- **Templates**: create/edit; preview renders inside the currently selected theme.
- **Security**: manage IP blocklist; set rate‑limit thresholds; review audit.
- **Users**: manage admin users and roles.
- **SMTP Test**: sends a themed multipart email using current settings.
